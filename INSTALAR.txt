1. Objetivo
  prueba
Este documento contiene las notas de instalacion de la aplicacion Expenses
version 2.5

2. Procedimiento

El procedimiento de instalación contiene 3 pasos (los cuales de detallan
a continuación), estos son:

    i) Poner en el código del aplicativo en una ruta publicada en Web
    ii) Crear la base de datos de MySQL y cargar el esquema de la base.
    iii) Configurar al aplicativo.
    
2.1 Poner en el código del aplicativo en una ruta publicada en Web

El aplicativo se encuentra dentro de un archivo comprimido con un nombre
similar a este: eexpenses_marti_20110418_164653.tgz (el nombre puede variar
puesto que depende de la fecha y hora a la que se genero el mismo).

Al descomprimir este archivo nos aparecera una estructura de directorios
como la siguiente:

    eexpenses_marti_20110418_164653 /
        eexpenses /
        eexpenses_marti.sql

La carpera "eexpenses" es el raiz de la aplicacion a instalar, esta contiene
los siguientes archivos:

            eexpenses/
            |-- admin
            |-- Connections
            |-- css
            |-- datos
            |-- flujos
            |-- forget.php
            |-- functions
            |-- images
            |-- imgs
            |-- index.php
            |-- inicial.php
            |-- INSTALAR.txt
            |-- lib
            |-- nbproject
            |-- php
            |-- procesos
            |-- reportes
            |-- sql
            `-- valida_usuario.php

Esta carpeta se debe de publicar en el servidor web.

2. Crear la base de datos de MySQL y cargar el esquema de la base.

Es necesario crear una base de datos de MySQL y cargar el script: eexpenses_marti.sql,
este script creara el esquema de la aplicacion asi como una serie de catalogos iniciales.

3. Configurar al aplicativo.

La configuracion del aplicativo se hace en el siguiente archivo:

  eexpenses/lib/php/constantes.php

opciones marcadas con una flecha son las que se tienen que ajustar)
